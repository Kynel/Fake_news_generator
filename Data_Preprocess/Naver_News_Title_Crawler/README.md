### How to use it

##### 1. run.py 를 실행.
##### 2. 결과값이 엑셀 파일로 출력됨. pandas 등의 라이브러리로 불러올 수 있음.

---------------------------------------------------------
코드 내에서 

'''python
csv_path = 'data.xls'
date = '20180101'
num = 1000
'''

이 부분의 값을 조정하면 됨
- csv_path : 저장될 파일 경로
- date : 이 날짜부터 하루씩 과거로 가면서 크롤링 함.
- num : 크롤링할 **분야별** 뉴스의 갯수. 여기서는 정치/국제의 두 섹션을 크롤링 하도록 설정해 두었으므로, 총 크롤링할 뉴스는 **num*2** 이다.
